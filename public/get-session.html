🔑 Ambil Session ID

Ikuti langkah berikut untuk mendapatkan Session ID TikTok kamu. Session ID diperlukan agar app bisa connect ke TikTok Live kamu secara otomatis.

1
Buka TikTok & Login
Tap tombol di bawah untuk buka TikTok. Pastikan sudah login dengan akun yang akan Live.
🎵 Buka TikTok.com
2
Copy Cookie dari TikTok
Setelah login di TikTok, kembali ke halaman ini.

Di Chrome HP, buka tab TikTok → ketuk address bar → ketik URL ini dan buka:

view-source:https://www.tiktok.com

Atau lebih mudah: tap ⋮ → Share → Copy link dari halaman profil TikTok kamu, lalu lihat panduan video di bawah.
3
Paste Cookie di sini
Copy semua isi cookie dari browser kamu, paste di kotak ini. Session ID akan diekstrak otomatis.
✅ Session ID Ditemukan
4
Cara Mudah Dapat Cookie di Chrome HP
1. Buka tiktok.com di Chrome → login
2. Tap address bar → tambah view-source: di depan URL
   Jadi: view-source:https://tiktok.com
3. Tekan Ctrl+F → cari sessionid
4. Copy nilai setelah sessionid= sampai tanda ;
5. Paste langsung di kotak di atas
⚠️ Penting: Session ID ini seperti password. Jangan bagikan ke siapapun selain dipakai di app ini. Session ID akan disimpan di server Railway kamu sendiri.
